Truck Sales Data Analysis Dashboard

Author: XIE LIDONG

Project Overview
A Streamlit-based interactive data analysis and visualization dashboard built for truck sales business. It loads sales data from Excel, computes core business KPIs including weekly and monthly sales, customer traffic, conversion rate, and provides multi-dimensional analysis by time, sales person, and truck type. The system supports daily, weekly, and monthly trend analysis, growth rate calculation, and automatic data caching for better performance.

Python, Streamlit, Pandas, Matplotlib, Data Analysis, Data Visualization, Excel Data Processing, KPI Monitoring, Time Series Analysis, Cache Optimization

Key Features
Automatic loading and cleaning of Excel sales data with error handling
Core KPI calculation including weekly and monthly total sales
Interactive multi-page dashboard with sidebar navigation
Multi-dimensional analysis: customer traffic, conversion rate, sales amount, sales quantity, salesperson performance, truck type distribution
Time-series analysis by day, week, month with growth rate tracking
Professional chart visualization with custom color scheme and layout
Data caching to improve loading performance
